English Channel Geostrophic Wind Metrics

will-faught

1 minute

The following plots show the sea-level pressure recorded in London, Paris and De Bilt over the last year and the geostrophic wind that has been derived from these three sites.

The plots are initially zoomed into the last 31 days but using the slider below the plots the timespan can be expanded up to the a year.

These values are calculated from the SLP readings taken at 1200 UTC.

These values are updated on a daily basis using values distributed via the Global Telecommunications System (GTS).
