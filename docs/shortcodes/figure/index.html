Paige provides a paige/figure shortcode for displaying content.

Basic #

Code:

{{< paige/figure >}}
Hickory dickory dock.<br>The mouse ran up the clock.<br>The clock struck one,<br>The mouse ran down,<br>Hickory dickory dock.
{{< /paige/figure >}}
